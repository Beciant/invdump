# invdump
Dumps inventory with augment decriptions to a text file (Ashita v4)

Save the included files in your Ashita addon folder. ex: Ashita-V4\addons\invdump
The file inclides the invdump.lua file and the assets folder containing the augment table for references.

Steps:
1. Type "/addon load invdump" in game.
2. Type "/invdump" to run the addon.
3. The addon will give a message "Inventory dumped to ..." when successful.
4. The file will save to your Ashita-v4\docs folder.
